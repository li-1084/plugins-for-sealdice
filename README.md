# plugins-for-sealdice
为海豹核心编写的插件。包括三角机构规则插件TA-triangle-agency，以及一些自创规则插件与娱乐插件。

说明：本项目中的插件均为独立的实现版本，目的是学习、练习编程。
本项目核心逻辑与规则实现均为开发者原创编写。
开发过程中，使用 [GitHub Copilot/Trae IDE]等工具辅助进行 
**第一版代码、重复性代码、语法补全、代码美化**等工作；
所有生成代码均经过人工评审、修改与验证。
